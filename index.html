____Bài 2: Chèn mã JavaScript vào thẻ "script"____

Bài này cho biết "script" có thể được tham khảo 3 cách sử dụng khác nhau:

1. với 1 URL đầy đủ(1 địa chỉ web đầy đủ) [ ví dụ: src="https://www.w3schools.com/js/myScript.js" ]

2. với 1 file path để liên kết đến link js [ ví dụ: src="/js/myScript.js" ]

3. Không có đừng dẫn nào [ ví dụ: src="myScript.js" ]

Nếu bạn Tap vào button "Thay đổi Text từ JS" ở dưới sẽ gán text với nôi dung khác trong JavaScript
